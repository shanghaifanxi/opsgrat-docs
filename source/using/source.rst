
资源管理
=============================

一、凭据管理
````````````````````

**1.1 查看凭据**
----------------------------------------

.. code-block:: vim

    点击“资源管理”下的“凭据管理”菜单，进入凭据管理页面，可以在该页面查看所有添加的凭据。
.. image:: ../_static/img/source_img/pingju1.png

----------------------------------------

.. code-block:: vim

----------------------------------------
